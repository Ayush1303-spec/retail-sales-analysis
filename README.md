# Retail Sales Data Analysis

This project analyzes a retail sales dataset using Excel pivot tables and data visualization.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization

## Key Insights
- Technology category generated the highest revenue.
- Certain regions significantly outperform others in total sales.
- Top-selling products contribute a large portion of overall revenue.
